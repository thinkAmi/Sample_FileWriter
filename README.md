Sample_FileWriter
========

Chrome拡張でFileSystemAPIを使ってローカルへファイルを作成するサンプルです。



動作確認環境
----------

* Windows7 x64
* Chrome 24.0.1312.52 m



セットアップ
----------

git clone後、「パッケージ化されていない拡張機能を読み込む」で追加することができます。
